1. 1.What is DOM and What is the purpose of DOM?

    2. Answer: Dom stand for document object model.The purpose of dom is it's connect to html document and make the webpage dynamic and interactive by javascript. Like we can change the font, size, color, background etc dynamically by using dom.

  2. 2.How will you select HTML elements using DOM? Name the DOM methods.

    3. Answer: I can select DOM in various way. There are five common ways to select dom:

    • 1. getElementById()
    • 2. getElementByClassName()
    • 3. getElementByTagName()
    • 4. querySelector()
    • 5. querySelectorAll()

  3. 3.What is event bubble?

    4. Answer: Bubble is a most important things in DOM. It happens when elements clicked or when an elements receive a events and bubble propagate to its parent. And it's propagate until it get to the root element.

  4. 4. What is the callback function and why you use it?

    5. Answer: A callback function is any reference to executable code that is passed as an argument to another piece of code. When we need to use one function to execute only after another function has completed execution then we use call back function. We can use this in javascript.
